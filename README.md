# 🎥 AI Video to Quiz

Convert videos or YouTube links into AI-generated quizzes.

## 🚀 Features
- Upload video OR paste YouTube link
- Automatic transcription
- Bloom’s taxonomy-based quiz generation
- Learning & Quiz mode
- Instant results

## 🛠 Tech Stack
Frontend: React + Tailwind  
Backend: FastAPI  
AI: Whisper / Gemini / NLP  

## ▶️ Run

### Backend
uvicorn main:app --reload

### Frontend
npm start
